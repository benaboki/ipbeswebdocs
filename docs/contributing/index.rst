Contributing
============

There are several ways you can get involved with developing the IPBES website e.g. ask a question, suggest improvements you would like to see.

Submit code
-----------

For developers who would like to suggest theme/appearance changes

- Pull requests for theme_ Add a link to the issue it fixes and a detailed description about your fix.
- Pull requests for views_ Submit a view export and a detailed description about your fix.

Connect
-------
- Follow @ipbes on Twitter_

.. _Pull requests for views: https://github.com/ipbes/ipbeswebdocs/tree/master/docs/views
.. _Twitter: https://twitter.com/IPBES
